# 项目简介
## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
- 管理员主页面作为系统控制中心
- 个人中心管理
- 项目负责人管理
- 项目类型管理
- 项目立项管理
- 项目变更管理
- 中期报告管理
- 结项报告管理
- 中期提醒管理
- 项目经费管理
- 绩效信息管理
- 绩效反馈管理
## 适用场景
- 高校科研项目管理
- 纵向科研项目监控
- 项目流程自动化管理
## 技术特点
- B/S架构，MVC模式
- 采用Java、SpringBoot、Vue技术栈
- 支持IDEA、JDK1.8、Mysql、Maven、Navcat开发环境
